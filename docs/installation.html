Installation

Two ways to get Python Chess running: grab a build for your system, or install it from source if you've already got Python.

Option A: download a ready-to-run build

This is the easiest path for most people. You don't need Python installed at all.

Go to the Releases page →

Grab the file for your operating system from the latest release, then:

  • Windows. Unzip the download and double-click PythonChess.exe.
  • macOS. Unzip the download and open PythonChess.app. macOS will throw up a security warning the first time, see Troubleshooting: opening the app on macOS for the quick, one-time fix.
  • Linux. Extract the archive and run ./PythonChess from a terminal.
Tip

Every release ships builds for Windows, macOS, and Linux, so you shouldn't need to build anything yourself.

Option B: install from source

Want to run it straight from the Python source instead? Maybe you're following along with updates on GitHub, or you want to poke around and change something. Either way, you'll need Python 3.10 or newer first.

git clone https://github.com/bottledpepsi/Python-Chess.git
cd Python-Chess

pip install -r requirements.txt

python main.py

That's really all there is to it. The game window should pop up right away.

Optional: a stronger computer opponent

Python Chess comes with its own built-in computer opponent, ready to go with no extra setup. If you want something noticeably stronger, and adjustable, you can add the free Stockfish chess engine instead. The app can download and set it up for you automatically right from the preferences screen, so there's nothing to configure by hand. Check out Features: Stockfish engine for more on this.

Building your own executable

Most people can skip this section entirely. It's really only useful if you've changed the source code and want to package it up yourself. Python Chess uses PyInstaller for this.

pip install pyinstaller

Linux:

pyinstaller --onefile --add-data "data:data" --name "PythonChess" main.py

macOS:

PyInstaller --onedir --windowed --icon=icon.ico --add-data "data:data" --name "PythonChess" main.py

Windows:

PyInstaller --onefile --windowed --icon=icon.ico --add-data "data;data" --name "PythonChess" main.py
